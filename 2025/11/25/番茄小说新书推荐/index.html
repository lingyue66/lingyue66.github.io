11/25. 《落榜后，我成贵妃面首，权倾天下》

（又名/曾用名：《娘娘，请卸甲！》）

  • 作者：红薯怪（番茄小说签约作家）

  • 状态：连载中

  • 字数：约 56.8 万字

  • 标签：历史军事 / 穿越 / 权谋 / 后宫

  • 核心类型：穿越历史 + 权谋 + 后宫 + 逆袭

  • 简介
    “我叫姬太初，穿越大梁皇朝，十年寒窗苦读，参加科举，做出锦绣文章却被冒名顶替，还被关进大狱。幸运遇到小时候进宫当太监的玩伴，从宫中回乡，帮贵妃娘娘物色面首。既然科举走不通，那我便换一种方式，权倾天下！”

6.《让你模拟人生，你模拟别人人生？》
  • 作者：某绝望的小小马甲（作家Lv.2）
  • 分类：玄幻脑洞
  • 状态：连载中
  • 字数：15.3万字
  • 评分：5.6
  • 阅读人数：3.4万人
  • 标签：玄幻脑洞、玄幻、穿越、系统、东方玄幻、无敌
  • 简介:

只要作者稳住就是一本至少精品的小说，跟一般模拟人生文不一样，作者的模拟过程简洁，没有长篇大论的模拟内容，可以使读者专注于故事而非模拟过程。这挂有点大，模拟别人人生，并且每日无需求免费三次，剧情符合番茄平台的短平快，工具人安排到位，一本纯爽文。

【有女主+有隐藏+有日常+有外挂+有脑子+杀伐果断+超级数值怪】
文笔还行，剧情一般，设定也不是那么合理，但是：这是一本 后宫文，作者N次进宫，上本书更改内容近400章节;有隐藏剧情，获取方式：阅读时长1小时以上+5星好评→截图审核→观看隐藏内容

数据表现：20万字+获9500+点评，评分9.5！

8.《西游：平账大圣在，我贪亿点咋了》
  • 作者：水咬纸
  • 状态：连载中，约 22.5 万字
  • 标签：玄幻 / 西游衍生 / 官场 / 基建 / 贪腐
  • 核心类型：官场西游 + 土木修仙 + 人民的名义（神仙版）+ 脑洞

简介:
讲述一个穿越者在天庭基建系统摸爬滚打三千年，利用“平账大圣”背锅的潜规则疯狂贪腐，最终在三界危机中挺身而出的故事。作品融合官场权谋、土木行业特性与西游神话设定，既以黑色幽默讽刺现实，又充满脑洞大开的创意情节。
